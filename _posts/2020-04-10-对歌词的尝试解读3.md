# 《多嘴》
-----

在这个系列的第一篇里我就提到了对应的[知乎问题链接](https://www.zhihu.com/question/266787151)。

这首是表达的对红色恐怖独裁的权贵资本主义的某个“你国”的讽刺和控诉。陈鸿宇自己写的词虽没有那么诗意，但也相当隐晦（在天朝的真理部监控下，这当然是不得不隐晦的）。

> 电视造机器 / 取灵魂 / 生产金币

$\uparrow$ 陈鸿宇自己的解释是“电视造（像）机器（一样的人）取（他的）灵魂 生产金币”。指天朝的各种毫无营养的脑残娱乐节目，剥夺人的独立深度思考能力。

> 岩​​石要取缔 / 新的楼 / 丝绸代替

$\uparrow$ 陈鸿宇自己的解释是“岩石（太坚硬了）要取缔 新的楼（用漂亮柔软的）丝绸代替”。有骨气的作者、记者等等都被安上种种“莫须有”的罪名封杀掉，当然要用软骨头的御用写手们代替了（周小平、花千芳等狗奴才）。这才是“新”时代啊！用丝绸造高楼，那可太稳如磐石了！！！！！厉害了你的国！！！！！！

> 吞掉壹把迷 / 罅隙中 / 琢磨痕迹

$\uparrow$ 陈鸿宇自己的解释是“吞掉壹把迷 罅隙中 琢磨（上头只言片语里的）痕迹”。各种高压线之下给人创新活力的空间是“罅隙”，一种戴着镣铐跳舞的感觉。

> 拆建之间 / 关闭墙面 / 别走近意义

$\uparrow$ 陈鸿宇自己的解释是“拆建之间关闭（VPN）墙面别走近意义（小心墙倒了砸着你）”。把头埋进沙子里的自欺欺人的所谓“文化自信”。把中国人挡在无数优秀网站之外, 仿佛在这些网站入口处设置了一道铁门, 上书八个大字: “华人与狗 不得入内”（来源：[编程随想的名言](https://program-think.blogspot.com/)）。

> 海浪解散歌声 / 和饮者的骄傲

$\uparrow$ 强劲的海浪就是一切，凡是它不解的，不愿意揭露的，会用尽一切手段压制，直至解散，只有骄傲的饮者才留下名字。所以，你所听到的歌声，只是海浪想让你听到的歌声。（来源：[如何评价陈鸿宇新歌“多嘴”？ - Tina Wu的回答 - 知乎](https://www.zhihu.com/question/266787151/answer/314571447)）。

> 情味低落爬进暗黄的注脚

$\uparrow$ 直播、快手等等娱乐方式层出不穷，在空耗时间的同时自娱自乐着，一步步爬向坟墓，这是最好的时代，也是最坏的时代。（来源同上）

> 蜷曲着伸展吧 / 生锈的布谷鸟  
喉咙里的火种长夜慢慢烧

$\uparrow$ 还是戴着镣铐跳舞的感觉。无论真理部的舆论控制是多么严厉而残忍，都要坚定初心，在这黑暗的社会环境中传播进步的抗争的火种，伸展到每一个角落。相信星星之火可以燎原。

> 饮者种的花 / 石碑下 / 沉默问答

$\uparrow$ 饮者应该是作者的自比。“种的花”应该影射的是“中华”，这是个很常见的规避敏感词审查方法。“石碑”是什么不知道。后面的内容都是作者的诘问。

> 微笑颜色家 / 替我选 / 那会美吗

$\uparrow$ 微笑/颜色/家。这是三件事物。大家可以想象一下这三个事物有没有被替选。家不能有人替我选。我想和谁是一家。就是一家。（来源：[如何评价或理解陈鸿宇《多嘴》歌词大意？ - WhiterJr的回答 - 知乎](https://www.zhihu.com/question/266931886/answer/418978401)）

> 又来新方法 / 让盛开 / 更安全吗

$\uparrow$ 中共总是擅长以“安全”之名，抹杀自由的空间。这个西方媒体已经揭示过无数次了。在毁人不倦的党妈妈眼里，让人民吸着奶嘴放弃独立思考当然是最安全的。

> 收起红布 / 叫醒门窗 / 也许谈意义

$\uparrow$ **这是意义最最明显的一句**。如果有人还想说前面的解释是我想多了的话，这句绝对狠狠打了他们的脸。

“红布”是崔健的《一块红布》。这块红布“蒙住我双眼也蒙住了天”，“我的手也被你攥住”，“将我的嘴堵住 / 我不能走我也不能哭”，让我“忘掉我没地儿住”只“看见了幸福”。

“门窗”是鲁迅在《呐喊》自序中的“铁屋子”。原文是
> 假如一间铁屋子，是绝无窗户而万难破毁的，里面有许多熟睡的人们，不久都要死了，然而是从昏睡入死灭，并不感到就死的悲哀。现在你大声嚷起来，惊起了较为清醒的几个人，使这不幸的少数者来受无可挽救的临终的苦楚，你倒以为对得起他们么?

“也许谈意义”中的“意义”应该就是“关闭墙面 / 别走近意义”中的“意义”。经过独立思考而不是人云亦云的言论才有意义。又或许是“若批评不自由 则赞美无意义”。
