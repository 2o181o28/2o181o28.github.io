# 背景

这是[对一些歌词的尝试解读：中](/2020/04/14/%E5%AF%B9%E6%AD%8C%E8%AF%8D%E7%9A%84%E5%B0%9D%E8%AF%95%E8%A7%A3%E8%AF%BB2/)的续集。内容是《与荒野》和《三旬》中的**部分**歌，合我口味的那部分。

# 《昨日的你》
-----

[知乎回答在此](https://www.zhihu.com/question/266233256/answer/307386191)。我觉得这个回答写得很棒，超级想直接复制一份放在这里。可惜不能转载。

这首是唐映枫写的词。另一个知乎回答里面说，就像是唐映枫对他的寄语。作为曾经的合作伙伴和难得有过心灵相通的朋友，字字真切如初。是对自己在乎的人最好的祝愿与谆谆告劝。歌词写得相当美，而且比较好懂。

> 当这世间如谜 / 歌声回答你  
原来无谓深意 / 只动听而已

$\uparrow$ 这句让人想起来了唐映枫自己说的“歌者一开口，歌词无意义”。这些深奥的歌词解释了许多难解的世事，作者不是故弄玄虚，只是为了与旋律与歌手的特点配合得更默契，更加打动人。

# 《多嘴》
-----

在这个系列的第一篇里我就提到了对应的[知乎问题链接](https://www.zhihu.com/question/266787151)。

这首是表达的对红色恐怖独裁的权贵资本主义的某个“你国”的讽刺和控诉。陈鸿宇自己写的词虽没有那么诗意，但也相当隐晦（在天朝的真理部监控下，这当然是不得不隐晦的）。

> 电视造机器 / 取灵魂 / 生产金币

$\uparrow$ 陈鸿宇自己的解释是“电视造（像）机器（一样的人）取（他的）灵魂 生产金币”。指天朝的各种毫无营养的脑残娱乐节目，剥夺人的独立深度思考能力。

> 岩​​石要取缔 / 新的楼 / 丝绸代替

$\uparrow$ 陈鸿宇自己的解释是“岩石（太坚硬了）要取缔 新的楼（用漂亮柔软的）丝绸代替”。有骨气的作者、记者等等都被安上种种“莫须有”的罪名封杀掉，当然要用软骨头的御用写手们代替了（周小平、花千芳等狗奴才）。这才是“新”时代啊！用丝绸造高楼，那可太稳如磐石了！！！！！厉害了你的国！！！！！！

> 吞掉壹把迷 / 罅隙中 / 琢磨痕迹

$\uparrow$ 陈鸿宇自己的解释是“吞掉壹把迷 罅隙中 琢磨（上头只言片语里的）痕迹”。各种高压线之下给人创新活力的空间是“罅隙”，一种戴着镣铐跳舞的感觉。

> 拆建之间 / 关闭墙面 / 别走近意义

$\uparrow$ 陈鸿宇自己的解释是“拆建之间关闭（VPN）墙面别走近意义（小心墙倒了砸着你）”。把头埋进沙子里的自欺欺人的所谓“文化自信”。把中国人挡在无数优秀网站之外, 仿佛在这些网站入口处设置了一道铁门, 上书八个大字: “华人与狗 不得入内”（来源：[编程随想的名言](https://program-think.blogspot.com/)）。

> 海浪解散歌声 / 和饮者的骄傲

$\uparrow$ 强劲的海浪就是一切，凡是它不解的，不愿意揭露的，会用尽一切手段压制，直至解散，只有骄傲的饮者才留下名字。所以，你所听到的歌声，只是海浪想让你听到的歌声。（来源：[如何评价陈鸿宇新歌“多嘴”？ - Tina Wu的回答 - 知乎](https://www.zhihu.com/question/266787151/answer/314571447)）。

> 情味低落爬进暗黄的注脚

$\uparrow$ 直播、快手等等娱乐方式层出不穷，在空耗时间的同时自娱自乐着，一步步爬向坟墓，这是最好的时代，也是最坏的时代。（来源同上）

> 蜷曲着伸展吧 / 生锈的布谷鸟  
喉咙里的火种长夜慢慢烧

$\uparrow$ 还是戴着镣铐跳舞的感觉。无论真理部的舆论控制是多么严厉而残忍，都要坚定初心，在这黑暗的社会环境中传播进步的抗争的火种，伸展到每一个角落。相信星星之火可以燎原。

> 饮者种的花 / 石碑下 / 沉默问答

$\uparrow$ 饮者应该是作者的自比。“种的花”应该影射的是“中华”，这是个很常见的规避敏感词审查方法。“石碑”是什么不知道。后面的内容都是作者的诘问。

> 微笑颜色家 / 替我选 / 那会美吗

$\uparrow$ 微笑/颜色/家。这是三件事物。大家可以想象一下这三个事物有没有被替选。家不能有人替我选。我想和谁是一家。就是一家。（来源：[如何评价或理解陈鸿宇《多嘴》歌词大意？ - WhiterJr的回答 - 知乎](https://www.zhihu.com/question/266931886/answer/418978401)）

> 又来新方法 / 让盛开 / 更安全吗

$\uparrow$ 中共总是擅长以“安全”之名，抹杀自由的空间。这个西方媒体已经揭示过无数次了。在毁人不倦的党妈妈眼里，让人民吸着奶嘴放弃独立思考当然是最安全的。

> 收起红布 / 叫醒门窗 / 也许谈意义

$\uparrow$ **这是意义最最明显的一句**。如果有人还想说前面的解释是我想多了的话，这句绝对狠狠打了他们的脸。

“红布”是崔健的《一块红布》。这块红布“蒙住我双眼也蒙住了天”，“我的手也被你攥住”，“将我的嘴堵住 / 我不能走我也不能哭”，让我“忘掉我没地儿住”只“看见了幸福”。

“门窗”是鲁迅在《呐喊》自序中的“铁屋子”。原文是
> 假如一间铁屋子，是绝无窗户而万难破毁的，里面有许多熟睡的人们，不久都要死了，然而是从昏睡入死灭，并不感到就死的悲哀。现在你大声嚷起来，惊起了较为清醒的几个人，使这不幸的少数者来受无可挽救的临终的苦楚，你倒以为对得起他们么?

“也许谈意义”中的“意义”应该就是“关闭墙面 / 别走近意义”中的“意义”。经过独立思考而不是人云亦云的言论才有意义。又或许是“若批评不自由 则赞美无意义”。

# 《金黄色》
-----

[知乎问题链接](https://www.zhihu.com/question/268114713)。

与《多嘴》一样是陈鸿宇自己作的词。批判更加隐晦，与《霓虹深处》很接近。从[豆瓣页面](https://music.douban.com/subject/30160258/)上可以了解到批判的对象是2017年北京的“清理低端人口”。“编曲中有绞带般断速，仿佛生活被生生切断。”

> 在冬天撕开的巨大 / 城市肚子里 / 掏出些炽热

$\uparrow$ “撕开的城市”、“炽热”明显是在隐喻那场造成19人死亡的大火。我原来的理解是一开始需要廉价劳动力时表现出的虚情假意与“宽容”，但是毕竟和“撕”这个动词不太匹配。

> 是你的是他的是谁的是大家的不不不是我的  
是你的是他的是谁的是大家的哦哦哦是我的  
......  
快来吧快走吧快说呀快闭上嘴巴都是我说的  
快来吧快走吧快说呀快闭上嘴巴说的都是你

$\uparrow$ 城市本来是属于大家的，本地的政府（或者是“城里酒肉金黄人”）既贪得无厌又妄图独占，所以先是热情地欢迎再是冷酷地驱赶外来人口，反复无常。“快说啊快闭上嘴巴”讽刺了真理部的舆论控制。

> 这是各位梦想城的 / 无限循环法和 / 升华的希望

$\uparrow$ “梦想城”当然就是北京了。“无限循环”说明这样的情况一直在发生。这首歌虽然中途崩坏了，到了最后却又恢复了开头的欢快情绪，正如一个循环。

> 城里酒肉金黄人的 / 黄金灯亮了 / 萤火虫就熄灭吧

$\uparrow$ “黄金灯”就是“低端人口”的辛勤付出满足了城里富人的贪欲。“萤火虫”是“低端人口”们微不足道的梦想。传达了一种卸磨杀驴的冷酷。

> 这是各位梦想城的 / 疼痛隐身术和 / 净化的希望

$\uparrow$ 继续装鸵鸟，看不见疼痛。“净化”这个词则让人想起来纳粹的所作所为。从这句歌词开始节奏忽然拖慢，伴奏的声音也变得诡异。然后是很长的一段停顿，与前面欢快的节奏形成了对比。

> 就在它里面 / 却又离它很远 / 冷点其实没关系

$\uparrow$ 被“切除”的“低端人口”虽然肉体仍在城市里，却已经被整个城市所排斥了。“冷点”隐喻的是强行切断了供暖和燃气。这在北京的冬天还怎么活啊。  
忽然想起来赵雷的《北京的冬天》里面写的“北京的冬天太冷 / 我没有足够的衣裳过冬”。扯得太远了。

其余的歌词意义都很明显，不需要我过多解释了。

# 《归还》
-----

[知乎问题链接](https://www.zhihu.com/question/268609890)。

按照一个回答里面所说的，这是在一专里就写好的曲子，是为《我来过》谱的曲。我猜测这个《我来过》就是刘昊霖《鱼干铺里》的那首《我来过》。后来唐映枫又给这首曲子填了《雨好》的词，却发现有更好的曲子所以搁置。填的第三次词就是这首《归还》。所以，《我来过》《雨好》《归还》这三首的词/曲自由搭配是毫无违和感的。

> 我来过 / 你的快乐与曾经  
你一挥手 / 我就告别成一道风景  
　　　　　　　　——《我来过》  
尤其四月 / 浮草间鱼儿跃起  
墙头破茧 / 枯木点绿铁罐叮咚有趣  
　　　　　　　　——《雨好》  
把星夜 / 归还给寒夜的魂  
草叶与车 / 归还给秋去时庄稼遍野  
　　　　　　　　——《归还》

这首歌仿佛是一个老人在退休时光中的平淡感悟，时光倒流一般，把自己一生的所有收获归还于自然与过往。风格比较接近一专。

> 草叶与车 / 归还给秋去时庄稼遍野

$\uparrow$ 没搞懂“草叶与车”和庄稼有什么关系，并且也没搜到相关的信息。我只能理解意象是丰收之后重归平静的农村生活。

> 灯与河川 / 归还给那洞悉孤往的眼

$\uparrow$ 我之前听歌的时候一直以为是“洞悉过往”。又是一个生僻的词“孤往”，搜索一下：独自前往，喻指归隐。显然我已经把“怀良辰以孤往，或植杖而耘耔”忘得一干二净了。我理解为已经看破红尘的隐士重新审视世间。

> 愿晚风起时 / 小屋仍是那间满月的 / 妈妈的屋

$\uparrow$ “小屋”是否就是众方纪呢？安于年岁，归还了一切，希望着将离开这个世界时，小屋还是儿时的小屋，美好的事物也仍像来时那么美好。

> 把野花 / 归还给钢铁大地  
酒与脊背 / 归还给那风尘里来的客

$\uparrow$ “钢铁大地”象征着现代文明，第一句话大概是说要在现代化的城市中保留一些原生态；第二句话中的“酒与脊背”让人想起江湖上的侠客，也许是说对朋友的真情。

# 《梦乡》
-----

[知乎问题链接](https://www.zhihu.com/question/337441160)。

《三旬》中我最喜欢的一首。陈鸿宇自己写道：“梦里挣扎过，醒来不得。亦如世事繁复，常求而不得。可唯贪恋，白日不得，全入梦。”然而《三旬》中的歌的风格已经与《浓烟下的诗歌电台》差得太远了，与普通的民谣区别不大，所以也就没必要特意解释歌词了。换句话说，弃坑了。

这首歌是“倦客”的一个好梦（美好的幻想）。实际世界中“玫瑰已有薄雾”（这句让人想起《你只是经过》中的“玫瑰暮色晚餐”），“半生已觉恍惚”。梦醒时就觉得失落怅然，无法再去寻味梦中趣处了。

# 《这不是理想的歌》
-----

《梦乡》和《夏的短篇》两首的旋律风格都与以往很接近，这首却是与别的陈鸿宇的歌差别很大。甚至可以夸张的说，旋律很俗很洗脑，比我听过的几乎所有民谣都更加接近广场舞音乐。歌词也很直白。然而这并不影响我对于这首歌的喜爱。

与专辑名称《三旬》连起来解释，我觉得就是对《理想三旬》这首歌的一个很棒的评价与注解。《梦乡》应该指的就是这首歌中的“面对挥别的梦的故事”。仍然是与《行歌》类似的在成年之后回首往昔的感慨。

# 《夏的短篇》
-----

顾名思义，歌词真的很短。和以前一样，堆砌了很多的意象。这些意象都是充满生机、希望与想象力的。

在夏天这个有希望的季节里，有收获的朋友在尽情歌唱。然而作者还是认为“孤独的马达声 / 如寂静中的耳语 / 最动听”。“旧皮卡”虽与周围的生机格格不入，这种孤独却也是一种别样的风景。

以这首歌的最后一句歌词作为结语吧，这是作者想对那些“有收获的朋友”说的：

> 你看那些现实四处散落  
你缺一些时间留给幻想

# 后记

拖了这么久，总算是写完了这一个系列。最大的收获就是了解了很多没人用的汉语词汇。其实还有很多别的歌的歌词非常深奥难懂的，比如赵照的《折断的骨头》（我在[APIO2019游记](/2019/05/16/APIOTHUSC219%E6%B8%B8%E8%AE%B0/#day3)里面引用过），但是由于不是陈鸿宇的，就不凑到这里来了。
